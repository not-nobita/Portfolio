🌟 Personal Developer Portfolio – Usama Noor

A modern, responsive, and visually stunning personal portfolio website built using HTML and CSS 🎨💻. This project showcases my journey as a Frontend Developer (Beginner) with a clean UI, smooth layout, and elegant dark luxury theme.

It includes a fully structured design with multiple sections:

🏠 Hero Section – Introduction with profile picture & call-to-action
👨‍💻 About Me – Short professional summary
⚡ Skills Section – HTML, CSS, JavaScript basics & DOM practice
📁 Projects Section – Featured projects with Live Demo 🚀 + Source Code 💻 links
📬 Contact Section – Social media & email links for easy reach

The design features a modern gradient background 🌌, smooth hover animations ✨, glassmorphism navbar, and a responsive layout for better user experience across devices 📱💻.

* Fixed glass-style navigation bar 🧭
* Project showcase with live & code links 🔗
* Clean and professional layout 💼
